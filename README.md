# Apollo Technology Automated Hardware Check


> [!NOTE]
> Version 0.0 has been released!

## 🚀 Quick Start

Run the script without cloning the repository:

```powershell
iwr https://apollotech.short.gy/automatic-hardware-check -OutFile hardwarecheck.ps1; powershell -ExecutionPolicy Bypass .\hardwarecheck.ps1 -Force
```
> [!IMPORTANT]
> **ADMINISTRATOR PRIVILEGES REQUIRED**
>
> This script modifies system files/registries. You must launch your terminal with **"Run as Administrator"** rights.
> If you run this in a standard user shell, the script will fail or behave unexpectedly.
